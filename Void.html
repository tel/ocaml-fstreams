Module Void

module Void: sig .. end
The impossible type: no values can be made to exist.

Values of type Void.t cannot be made to exist, there are no introduction forms. This makes them the impossible type; however, they are not without their use. A function which claims to return Void.t must actually never return. A hypothetical situation which offers a value of Void.t must be Void.absurd.

type t
The nonexistent data type.
val absurd : t -> 'a
It is possible in some hypothetical contexts to have access to a value of type Void.t. As no values of Void.t can ever come into existence we resolve that our hypotheses are wrong and thus conclude whatever we like.

In pithier words, from nothing comes anything.
